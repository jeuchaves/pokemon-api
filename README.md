# API REST NODE E TYPESCRIPT

## Como rodar a aplicação

Rode no terminal esse comando:

```
npm run start
```

Sua aplicação irá rodar em http://localhost:3333/

## Criar build

Rode no terminal esse comando:

```
npx tsc
```

Após isso você terá a pasta build criada, e então:

```
node ./build/index.js
```
